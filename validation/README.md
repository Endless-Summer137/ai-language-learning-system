# Validation

This folder records real validation work for the AI Language Learning System.

The goal is not to make the project look complete. The goal is to record what has actually been tested, what worked, what failed, and what still needs design revision.

## What belongs here

- Real test logs
- Model behavior observations
- Session boundary failures
- Runtime loading issues
- Report and anchor quality checks
- Cross-model comparison notes

## What does not belong here

- Unverified demo scripts
- Personal presentation notes
- Private planning drafts
- Unrun assumptions presented as results

## Recommended structure

```text
/validation
  README.md
  validation-log-template.md
  test-index.md
```

Future validated records can be added as separate files when a test is completed.
